## Valorant Champions Tour (VCT) Data Analysis and Visualization

Welcome to my Valorant data analysis repository! This project dives into professional match data from the Valorant Champions Tour (VCT) to uncover the statistical patterns that define successful team compositions, map metas, and individual agent performance. As a high-ranked player, I know how easy it is to fall into the trap of judging impact solely by the killfeed while neglecting role-specific responsibilities. I built this project to bridge that gap by translating complex pro-level data into accessible visuals that help players assess their gameplay more holistically.

The visualizations in this repository are created using R and explore thousands of professional matches to reveal what truly contributes to winning.

Visualizations Included:
- **Agent Role Importance (Radar Plots)**: Highlights the unique responsibilities of Duelists, Initiators, Sentinels, and Controllers by mapping their average ACS, Kills, First Engagements, Assists, and Orb Gain.
- **Correlation Matrix of In-Game Stats**: Reveals how different in-game metrics (like ADR, HS%, and KAST) influence each other and impact overall success.
- **Average Statistics per Agent (Data Table)**: Provides specific statistical benchmarks (like average damage and first engagements) for every agent based on consistent pro-level performance.
- **Meta Team Compositions**: Showcases the top five agent synergies and compositions used by VCT pros across different maps.
